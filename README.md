# Widgets
## Descrizione:
Progetto di test per studio del framework di Avalonia Ui.

**Al momento l'applicativo tratta solo di:**
- Gestione cambio view tramite una sidebar con view molto basilari
- Gestione delle icone per sezioni sidebar
- Inizio di una struttura MVVM
